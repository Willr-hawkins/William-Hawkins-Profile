Binary Beans

Binary Beans is the 4th project I made for my course with Code Institute. The wesbite is for a made up coffee shop, where the users can learn more about the coffee shop, find out about up and coming events, and see what is on the coffee shops menu. The technologies used in this project were HTML, CSS, Bootstrap CSS Framework, and Django Framework. The key funcitonalities of this application are the following:

  • Sign up for an account, aswell as log in and sign out.
  • Request a private event at the shop.
  • Edit and delete private event requests.
  • Leave a review on the website.
  • Leave review on products.
  • Edit and delete reviews.
  • Leave a like on other users reviews.

A multitude of tools were used to help develop this application, the main tools used where GitHub for the app repository, GitPod a cloud based development enviroment, and ElephantSQL a potgres database.
