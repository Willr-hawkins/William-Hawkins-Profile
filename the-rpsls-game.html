The R.P.S.L.S Game

This game is a player V computer single player game, based off the traditional game many have played, rock, paper, scissors. The additional two moves come from the version of the game in the Big Bang Theory show, with the two additonal moves being lizard, and spock. Some key features of this application are the following:

  • User input feedback.
  • Real time score count.
  • First to reach a score of 5 wins.
  • Computer disicions are completely randomized.
  • Messages to show who won each round.
