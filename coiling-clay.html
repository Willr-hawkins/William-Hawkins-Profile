Coiling Clay

Coiling Clay was my 5th and final project for my Diploma in Full Stack Software Development with Code Institute. The project is a Django based project for a made up Ceramics online store. The technologies used to develop this application were HTML, CSS, JavaScript, Python, Boostrap CSS Framework, and Django Framework. The key funcitonality of the application are the following:

  • Adding products to cart and editing cart contents.
  • ability to checkout and easily make a payment.
  • Create a user account.
  • Save personal details within the account section, along with viewing purchase history.
  • Leave review on products.
  • Commenting on other user's reviews.
  • Edit and delete reviews and comments if the user is the owner.
  • Create wishlists and add products to wishlist's.

A multitude of tools were used to help develop this application, the main tools used where GitHub for the app repository, GitPod a cloud based development enviroment, and ElephantSQL a potgres database.

Along side these tools, Stripe was also used. Stripe is a service that allows compaines to accept payments from clients/cusomters. For this project only the test mode was used, which enables the ability to test that payments are working before going live to real customers.

Visit the Stripe website!

AWS (Amazon Web Services) was used during the deployment of this application. Two services were used the first being Amazon S3, this service allows the developer to create a bucket to hold all static and media files for the website. The second service used is IAM (Identity and Managment) this allows the developer to manage the workload and workforce of the application.
